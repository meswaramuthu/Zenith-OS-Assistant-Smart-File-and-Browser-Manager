# Zenith-OS-Assistant 🚀

**Built for the SVCE Hackathon**, Zenith is a voice- and text-controlled OS assistant that uses AI to simplify file organization and browser tab management.

### Key Features ✨
- **Smart File Management**: Automatically organize, move, and rename files by type or date.
- **Browser Control**: Open, close, and manage tabs/bookmarks via natural language.
- **Multi-Modal Input**: Supports both voice and text commands.

### Tech Stack 🛠️
- **Backend**: Python (Flask)
- **AI/NLP**: OpenAI, spaCy, SpeechRecognition
- **Automation**: OS/shutil, PyAutoGUI, Selenium

### Quick Start ⚡
```bash
git clone https://github.com/sujitlaware1809/Zenith-OS-Assistant-Smart-File-and-Browser-Manager.git
cd Zenith-OS-Assistant-Smart-File-and-Browser-Manager
pip install -r requirements.txt
python app.py
```
